Anotações áudios

Assim como a tag <picture> para utilizarmos áudios é necessário usar a tag <audio>

Referências: https://www.w3schools.com/html/html5_audio.asp

Atributos:

  • controls: adiciona controles de mídia(play, pause e volume);
  • src: indica o áudio a ser tocado e o type o tipo (ogg, mp3 e wav);
  • preload: indica a forma que o arquivo será carregado
    • metadata: apenas informações (nome, tamanho, tempo);
    • auto: carregará tudo;
    • none: não carregará nada até o usuário apertar o play.
  • loop: o áudio será repetido infinitamente.
  • autoplay: indica a reprodução do áudio assim que a página é carregada;
Formatos e tipos de mídia:
  • File format: ogg - Media Type: audio/ogg
  • File format: mp3 - Media Type: audio/mpeg
  • File format: wav - Media Type: audio/wav
Exemplo de algoritmo de áudio: 
    
    
<audio controls autoplay  loop> 
    <source src="caminho-do-arquivo.extensão" preload="metadata" type="audio/ogg">
    <source src="caminho-do-arquivo.extensão" preload="metadata" type="audio/wav"> 
    <source src="caminho-do-arquivo.extensão" preload="metadata" type="audio/mpeg">    
</audio>
