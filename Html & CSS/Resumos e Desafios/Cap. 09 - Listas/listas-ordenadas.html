Listas Ordenadas

Listas ordenadas são utilizadas quando a ordem dos componentes importam, são representadas por: <ol>(ordered list) e seus termos são representados por: <li>(list item).

Tipos de listas

Por padrão as listas ordenadas utilizam o type 1, ou seja, ordenam por numeração crescente.
  1. Type 1(Padrão) → Listas Numeradas: 1, 2, 3, 4
  2. Type A → Listas Alfabéticas (Maiúsculo): A, B, C, D
  3. Type a → Listas Alfabéticas (Minúsculo): a, b, c, d
  4. Type I → Listas em Algarismo Romano (Maiúsculo): I, II, III, IV
  5. Type i → Listas em Algarismo Romano (Minúsculo): i, ii, iii, iv

Exemplo de lista ordenada padrão

  1. Item 1
  2. Item 2
  3. Item 3

Listas Não Ordenadas

Listas não ordenadas assim como o nome sugere são utilizadas quando a ordenação não é um fator importante,tais listas são representadas por: <ul>(unordered lists) e seus itens também são representados por: <li> assim como nas listas ordenadas

Exemplo de lista não ordenada padrão

  • Item 1
  • Item 2
  • Item 3

Ao contrário das listas ordenadas, o parâmetro type nas UL alteram a forma do "asterísco"

Tipo "circle"

  • Item 1

Tipo "disc"

  • Item 1

Tipo "square"

  • Item 1

Lista de definições

Assim como o nome sugere esta lista funciona como uma espécie de dicionário, onde: <dl>(definition list), <dt>(definition term) e <dd>(definition description)
