Anotações Imagens

Atualmente os sites são acessados em diferentes dispositivos e se faz necessário "edições" para que os conteúdos sejam mostrados adequadamente em cada dispositivo, ou seja, é necessário responsividade.

Utilizando a tag <picture> e a tag <source> é possível configurar qual imagem deve aparece de acordo com o tamanho da página.

  • o atributo media="(max ou min-width: ) definimos o tamanho maximo a ser considerado para carregar a imagem adequada.
  • o atributo srcset indicará a imagem que será carregada quando o tamanho indicado no atributo media for atingido
  • o atributo type indica o media type do arquivo (.png, .jpeg, .mp4)

Algoritmo de exemplo 


    <picture>
        <source media="(max ou min-width: tamanho)" srcset="caminho para o arquivo" type="image/png" >
        <img src="" alt="">
    </picture>
