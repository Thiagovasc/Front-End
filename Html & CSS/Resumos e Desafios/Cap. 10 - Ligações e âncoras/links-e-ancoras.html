Links e Âncoras

Hiperlinks devem ser criados usando a tag <a> e tem como principal atributo o href=""

Exemplo: <a href="meusite.com">Entre no meu site</a>

O atributo hreflang="" é utilizado para indicar o idioma do site, auxiliando assim as ferramentas de tradução

Outro atributo muito importante é o target, responsável pela forma que o link será aberto, na mesma, outra aba, etc...

  • _blank: outra aba
  • _self: mesma ba
  • _top: desfaz os frames e abre o destino no navegador por completo
  • _parent: similar ao uso do top

O atributo rel é utilizado para indicar a natureza do link

  • next: indica que é a continuação do doc atual
  • prev: indica que é a parte anterior do doc
  • author: link do autor do doc
  • external: link para outro site
Exemplo <a href="" hreflang="" rel="">Nome link.. </a>

Links pra download

Para inserir links de download é necessário utilizar a tag <a> com o atributo download="link"

Exemplo: <a href="meulivro.pdf" download="livro.pdf" type=""> Download</a>
