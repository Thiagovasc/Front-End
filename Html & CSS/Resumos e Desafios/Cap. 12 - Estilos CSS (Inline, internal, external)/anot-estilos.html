CSS - Cascade Style Sheets

A beleza do estilo e da organização

O conteúdo do seu site é o núcleo, a parte principal não é sábio ignorar o design, o estilo, sites mais bonitos, organizados, atraentes atraem mais público ou ao menos não espanta os que já acharam seu site.

  • Inline style;
  • Internal style;
  • External style.

A 1ª técnica de aplicar estilos é a maneira CSS Inline Style, tal maneira é mais simples, não é errada nem obsoleta até então, mas é a mais simples e um tanto quanto ineficaz, esta forma consiste em aplicações pontuais, assim como foi feito no desafio 09

Exemplo: <body background="blue"></body>

A 2ª técnica de aplicar estilos é o Internal style, onde se utiliza a tag <style></style> no head do documento html.

Exemplo:

            
    <style>
        body {
            background-color: blue;
            font-family: Arial, sans-serif, Helvetica;

        }
    </style>
Atenção: a tag style deve estar dentro do head do seu documento, a mesma pode até funcionar estando dentro de outra tag, mas não estará em conformidade com as determinações da w3c

A 3ª técnica de aplicar estilos é a External styles, a mais versátil, desta forma as formas de estilo não se misturam diretamente ao código html e são interligadas a partir da tag link que deverá ser colocada no head

Obs: Coloca-se @charset "utf-8"; para evitar problemas com acentuação, comentários são feitos assim em css /* */
