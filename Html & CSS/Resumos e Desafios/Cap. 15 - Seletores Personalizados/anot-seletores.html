Seletores Id(#) e Class (.)

Ao criar conteúdo em html podemos personaliza-los com CSS utilizando os seletores padrões (body, p, h1), mas também há a possibilidade de usar seletores personalizados.

O seletor id só pode ser utilizado uma vez por documento html, já o class pode ser utilizado multiplas vezes.

Um exemplo de utilização para tais seletores é classificar o <h1> como id "titulo principal" e os <h2> como tópicos ou subtópicos, assim a personalização em css se torna mais ágil e simples.

Dica: tag.nome da classe + enter resulta em: <p class="principal"></p>
O mesmo acontecerá para id, se substituir o ponto pela #

É possível atribuir mais de uma classe a uma tag, basta separa-las por espaço <p class="destaque principal"></p>

Este tipo de agrupamento é muito útil, pois, as vezes você pode desejar que tags iguais possuam formatações diferentes, assim como esta tag p aqui.

Sendo a técnica externa, a mais versátil e a que deixa seu código mais bonito, porém é possível utilizar todas ao mesmo tempo, estes estilos serão somados, caso haja duplicidade nas configurações, o estilo inline prevalecerá perante aos outros.

Pesos de estilos

  • Mais baixo: seletores de tipo e pseudo-elementos
  • Baixo: classes, pseudo-classes e atributo
  • Moderado: seletores de id
  • Alto: inline

Pseudo-classes em CSS

Uma pseudo-classe em CSS é uma palavra atribuida a declaração de um seletor após o sinal de :, a mesma se refere ao estado de um elemento.

Exemplos: :hover, :before, :after, :first-letter...

O :hover por exemplo representa o estado quando o mouse passa por cima do elemento exemplo

Pseudo-elementos

Os pseudo-elementos interferem no conteudo e são atribuidos utilizando "::", exemplo: a::after

Resumo do capítulo

  • . = class
  • # = id
  • : = pseudo-classe
  • :: = pseudo-elemento
